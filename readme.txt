Descrição do projeto - Português


Este projeto deseja facilitar o processo de copia de arquivos de uma maquina host para varias maquinas focando na agilidade e ganho de tempo e deve ser utilizado primordialmente por administradores de rede e equipes de Suporte entre suas caracteristicas destacan-se

-Interface simples.
-Arquivo de configuração em XML podendo ser editado no excel.
-Uso de multithread para as copias.
-Gerar logs de erros com problemas encontrados
